# ☕ NOVA Coffee Shop

A modern, mobile-first coffee ordering web application.

## Features

- 📱 Mobile-first responsive design
- 🛒 Full cart & ordering system
- 🎨 Clean blue & white theme
- ⚡ Single HTML file - no dependencies

## Pages

- `index.html` - Mobile order app
- `landing.html` - Landing page (dark futuristic theme)
- `pos.html` - Point of Sales system

## Live Demo

Open any HTML file directly in browser.

## Tech Stack

- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
